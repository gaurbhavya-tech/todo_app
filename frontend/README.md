# 🚀 Smart Todo App

A full-stack Todo application with file upload feature built using React, Node.js, Express, and Multer.

---

## 📌 Features
- ✅ Add tasks
- 📁 Upload files with tasks
- 🗑️ Delete tasks
- ⚡ Fast and simple UI

---

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- File Upload: Multer
- API Calls: Axios
